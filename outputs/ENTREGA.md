# Entrega do Receitagram

Projeto criado na pasta da conversa com:

- App Android em Java: `app/`
- API REST Java: `backend/`
- Documentação principal: `README.md`
- Documentação da API: `docs/API.md`

## Primeiro teste recomendado

1. Rode a API:

```bat
cd backend
run-api.bat
```

2. Abra a pasta raiz do projeto no Android Studio.
3. Execute o app em um emulador Android.
4. Entre com `ana@demo.com` e senha `123456`, ou crie uma nova conta.

## Observação de validação

O ambiente desta conversa bloqueou execução de compiladores e processos externos, então a compilação não pôde ser feita automaticamente aqui. A estrutura dos arquivos e os pontos essenciais do projeto foram conferidos por leitura dos arquivos gerados.
